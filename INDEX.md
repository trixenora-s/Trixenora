## üîë Key Information Quick Reference

### Installation Command
```bash
./install.sh && ./dev-setup.sh
```

### Start Services
```bash
docker-compose up -d
```

### Start Development
```bash
# Terminal 1: Backend
cd server && npm run dev

# Terminal 2: Frontend
cd client && npm start
```

### Access Points
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000/api
- MongoDB: localhost:27017

### Test User (After Signup)
- Username: any username
- Email: any email
- Password: any password (min 6 chars)

### Default Admin (MongoDB)
- Username: admin
- Password: password

## üöÄ Quick Commands

```bash
# Install all dependencies
./install.sh

# Setup development environment
./dev-setup.sh

# Start Docker services
docker-compose up -d

# Start backend (Terminal 1)
cd server && npm run dev

# Start frontend (Terminal 2)
cd client && npm start

# Stop services
docker-compose down

# View logs
docker-compose logs -f

# Connect to MongoDB
docker exec -it trixenora-db mongosh
```
